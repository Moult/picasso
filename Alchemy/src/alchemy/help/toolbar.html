Alchemy Toolbar

The Alchemy toolbar is hidden automatically and can be displayed by moving the cursor into the top area of the canvas. The idea being to remove as much distraction as possible from the creative process.

  • Style

    Toggle between drawing with a line or solid fill.

  • Clear

    Clear the entire Alchemy canvas.

  • Line Weight

    Increase/Decrease the line weight.

  • Black/White

    Toggle between black and white

  • Transparency

    Adjust the transparency

  • Create

    The create dropdown menu contains all of the currently loaded 'create' modules. Create modules construct shapes in some way, either generatively or with user input from the cursor. Only one create module can be active at a time.

  • Affect

    The affect dropdown menu contains all of the currently loaded 'affect' modules. Affect modules change existing shapes or alter the drawing environment in some way. Multiple affect modules can be selected and may cause unpredictable results (that is a good thing!).
