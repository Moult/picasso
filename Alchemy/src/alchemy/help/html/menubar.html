Alchemy Menubar

On Windows the Alchemy menubar is hidden automatically and can be displayed by moving the cursor into the top area of the canvas. On Mac OSX the menubar is displayed in the top menubar.

  • File
    • New

      Clears the canvas - functionally identical to the clear button in the toolbar.

    • Export

      Exports the canvas as a single page PDF file.

    • Page Setup

      Define the page settings for printing.

    • Print

      Print the canvas.

  • View
    • Fullscreen

      Enter fullscreen mode. To exit fullscreen mode, hit the ESCAPE key.

  • Session

    The session menu controls how a Alchemy 'session' is recorded. Unlike the normal export comman the session menu creates a multi-page PDF file with pages saved either automatically or at timed intervals.

    • Toggle Recording

      Turn on/off recording of a session. To correctly create the PDF file, the session must be explicitly started as well as ended using the 'Toggle Recording' menu item.

    • Save Page

      Manually save a page to the session PDF file.

    • Save Page & Clear

      Manually save a page to the session PDF file, then clear the canvas.

    • Record on Startup

      Begin recording a new session when Alchemy is started.

    • Record Interval

      Set a timed interval to the save pages to the session PDF file. The default setting, Manual, turns the timed interval off and will only save a page with either of the 'Save Page' commands. When set to a specific time, pages will be saved continuously according to that time.

    • Auto Clear Canvas

      When turned on this setting will automatically erase the canvas after it has been saved using the record interval.

    • Set Session Directory...

      Set the directory where the PDF file will be saved to. The file is created automatically and saved to this directory with a name such as "Alchemy-2008-02-07-50-37.pdf".

  • Switch

    The switch menu allows you to quickly open the current Alchemy canvas in another user-defined application. The idea is to be able to quickly continue working in a more 'conventional' software application without loosing the creative 'flow'.

    • Switch Vector

      Open the current Alchemy canvas as a vector graphic in an external application.

    • Switch Bitmap

      Open the current Alchemy canvas as a bitmap graphic in an external application.

    • Set Vector Application...

      Choose an external application to open the Alchemy canvas in vector format.

    • Set Bitmap Application...

      Choose an external application to open the Alchemy canvas in bitmap format.

    • Alchemy Forum

      Open the Alchemy forum in the default browser.

    • About Alchemy

      Open the about window (Windows only).
